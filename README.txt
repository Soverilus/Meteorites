Maya Models contains the models created by Asif Juran and Ryan Williams for the Meteorites project
Meteor contains the Project folder for unity - including all the scripts, prefabs, and assets used to create the game
MeteoritesGame contains the built game as it stands.

The game requires at minimum two controllers, but there is a public bool under MeteoriteHandler in the Arena scene to force the game to start
(make sure to join the game before it does by pressing the A button).
